REST API
--------

### GET/POST td_attr

Returns term dependency and term attractiveness.

#### Resource URL

http://wordcolony.satomacoto.com/td_attr

#### Parameters

- text (reqired)
    - up to 10,000 characters
- lang
    - en/ja
- callback
